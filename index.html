เชื่อมต่อ Google Sheets

ข้อมูลทั้งหมดจะถูกเก็บใน Google Spreadsheet ของคุณ
ทีมทุกคนเห็นข้อมูลเดียวกัน Real-time

1
ไปที่ script.google.com → สร้างโปรเจกต์ใหม่ → วางโค้ดจากไฟล์ Code.gs
2
แก้ไข SPREADSHEET_ID ในโค้ด → Run function setupSheets() ครั้งแรกเพื่อสร้าง Sheet
3
กด Deploy → New Deployment → Web App
ตั้ง Execute as: Me · Who has access: Anyone
4
Copy Web App URL แล้ววางด้านล่าง
ใช้ LocalStorage แทน (ข้ามไปก่อน)
